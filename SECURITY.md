# Security Policy (VDP)

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | ✅ Active          |
| 1.1.x   | ❌ End of Life     |
| < 1.0   | ❌ End of Life     |

## Reporting a Vulnerability
Security is a pillar of digital sovereignty. If you discover a vulnerability:
1.  **Do not open a public Issue.**
2.  Submit a detailed report to: `security@your-domain.com`.
3.  We will acknowledge receipt within 24 hours and provide a remediation timeline within 7 days.

## Coordinated Disclosure
We follow a Coordinated Vulnerability Disclosure (CVD) process. Once a patch is deployed, a formal Security Advisory will be published to inform the community while protecting critical environments.
